# test-paytend

To install dependencies:

```bash
bun install
```

To prepare project:

```bash
add your keys to ./keys/

paytend_public.pem
paytend_private.pem
my_public.pem
my_private.pem
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.35. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
